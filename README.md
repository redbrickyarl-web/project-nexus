# Project Nexus

God-Tier Sovereign High-Performance Rust Architecture

## Features
- Cranelift JIT compilation
- Zero-copy CosmicQueue
- WASM Sandbox with Wasmtime
- Post-Quantum Crypto (Kyber + Dilithium)
- GPU acceleration with WGPU
- BFT DAG consensus with libp2p
- Docker multi-node setup

## Quickstart
cargo build --release -p nexus
